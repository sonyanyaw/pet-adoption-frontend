# Fluffy — Сайт приюта для животных

Современный анимированный маркетинговый сайт для приюта по усыновлению животных. Создан на основе Next.js 15, React 19, TypeScript, Tailwind CSS 4 и Framer Motion.

> English version: [README.md](./README.md)

---

## Описание

Fluffy — это одностраничный сайт, созданный для того, чтобы помочь животным найти заботливый дом. Сайт демонстрирует животных, доступных для усыновления, рассказывает о том, как помочь приюту через пожертвования или волонтёрство, отвечает на часто задаваемые вопросы и предоставляет контактную информацию — всё это в рамках плавного и отзывчивого интерфейса.

### Разделы сайта

| Раздел | Описание |
|---|---|
| **Hero** | Анимированный экран-приветствие с изображениями питомцев и кнопкой призыва к действию |
| **О нас** | Миссия приюта и ключевые показатели |
| **Питомцы** | Карточки животных, доступных для усыновления, с hover-эффектами |
| **Помочь** | Призыв к пожертвованиям и волонтёрству |
| **FAQ** | Раскрывающийся аккордеон с ответами на популярные вопросы |
| **Контакты** | Секция контактов со ссылками на социальные сети |

## Начало работы

### Требования

- Node.js 18+
- npm, yarn, pnpm или bun

### Установка

```bash
git clone https://github.com/sonyanyaw/pet-adoption-frontend.git
cd pet-adoption-frontend
npm install
```

### Разработка

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере. Страница автоматически обновляется при изменении файлов.

### Сборка для продакшена

```bash
npm run build
npm start
```

### Линтинг

```bash
npm run lint
```

## Развёртывание

Самый простой способ развернуть проект — [Vercel](https://vercel.com): подключите репозиторий GitHub, и сайт будет автоматически деплоиться при каждом пуше в ветку `main`.

Для других платформ (Netlify, Railway, собственный сервер) выполните `npm run build` и раздайте содержимое директории `.next`. Подробнее см. в [документации по деплою Next.js](https://nextjs.org/docs/app/building-your-application/deploying).
